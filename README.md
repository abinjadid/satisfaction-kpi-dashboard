# لوحة معلومات مؤشر الرضا عن الخدمات الاستشارية

لوحة معلومات تفاعلية (KPI Dashboard) لقياس **مؤشر الرضا عن الخدمات الاستشارية**،
بواجهة عربية كاملة واتجاه RTL وتصميم حكومي مؤسسي حديث مع دعم الوضع الليلي.

## التقنيات

- **React 18 + TypeScript**
- **Vite** أداة بناء سريعة
- **Tailwind CSS** + مكوّنات بأسلوب shadcn/ui
- **Recharts** للمخططات
- **Framer Motion** للحركات
- **React Hook Form + Zod** للنماذج والتحقق
- **html2canvas + jsPDF** لتصدير PDF/PNG
- **Lucide Icons** للأيقونات
- خط **IBM Plex Sans Arabic**

## التشغيل

```bash
npm install
npm run dev
```

ثم افتح الرابط الظاهر في الطرفية (عادة <http://localhost:5173>).

### أوامر أخرى

```bash
npm run build     # بناء نسخة الإنتاج
npm run preview   # معاينة نسخة الإنتاج
npm run lint      # فحص الأنواع (TypeScript)
```

## أبرز الميزات

- بطاقات مؤشرات (KPI) متحركة مع حلقات تقدّم ومؤشرات اتجاه (↑↓) ومقارنة بالشهر السابق.
- مخططات احترافية: Gauge، خطي، أعمدة، دائري، راداري، وأعمدة أفقية.
- ملخص استنتاجات تلقائي (Insight Cards) يُحسب من البيانات.
- تصفية متقدّمة (السنة، الشهر، نوع الخدمة، الجهة، بحث) مع إعادة تعيين.
- إضافة استجابة عبر نافذة حوارية مع تحقق كامل، وإعادة حساب كل المؤشرات لحظياً.
- تصدير **PDF** بحجم A4 عمودي، و**PNG**، و**CSV**، و**Excel**، إضافة إلى **الطباعة**.
- وضع ليلي، حالات تحميل (Skeletons)، حالة فارغة أنيقة، وتنبيهات (Toasts).
- جدول أحدث الاستجابات مع فرز، وقوائم الخدمات الأعلى/الأقل أداءً.

## التخزين

تُحفظ البيانات حالياً في **Local Storage** عبر طبقة خدمة نظيفة
(`src/services/dataService.ts`) مبنية على واجهة `IDataService`،
مما يسهّل استبدالها لاحقاً بـ API دون تعديل بقية الكود.

## بنية المشروع

```
src/
├── components/
│   ├── ui/          # مكوّنات أساسية (Button, Card, Dialog, ...)
│   ├── dashboard/   # مكوّنات اللوحة (KPI, Filters, Insights, ...)
│   ├── charts/      # مكوّنات المخططات
│   └── forms/       # نماذج الإدخال
├── hooks/           # خطافات مخصّصة (useResponses, useTheme, ...)
├── services/        # طبقة البيانات والتخزين
├── types/           # واجهات TypeScript
└── utils/           # الحسابات، التنسيق، التصدير، الثوابت
```

> الشعار في الترويسة والتقرير هو عنصر نائب (placeholder) يمكن استبداله بشعار الجهة.
